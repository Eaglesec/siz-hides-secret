FAQs

What is Digi-Cloak?

Digi-Cloak is an online web tool to hide a secret message in the image using cryptography and steganography.

How does it work?

Message is encrypted into ciphertext using AES with a 256-bit key. This key is derived from a password specified by you. Encrypted ciphertext is hidden in the image using the LSB (Least Significant Bit) technique in such a way that the image is unaffected. The image is also provided by you.

What information is retained?

No information is retained, all processes happens at client-side only.

How secure is my message?

Your message, password and image are never transmitted, all process takes place client-side in your browser.

I misplaced my password, is my message recoverable?

No, that is the point. Your message, password and image are never submitted beyond your browser. If the password or image is misplaced, the original message is gone forever.

Is there a limit to the length of message to be hidden?

Yes, message length should not greater that 4 times the number of pixels (height x width) in specified image.
